## 5 色轮交互组件

### 5.1 目标

创建基于Canvas的HSL色相环，支持拖拽选取基色。

### 5.2 技术要求

- **尺寸**：280px × 280px（包含Canvas和外围空间）
- **Canvas元素**：260px × 260px
- **实现方式**：原生Canvas API + JavaScript事件监听

### 5.3 视觉要求

1. 绘制完整的HSL色相环（360度），圆心为白色向外径向渐变至纯色（饱和度100%，明度50%），圆周边界清晰
2. 色轮外围有2px边框，颜色#E5E7EB，圆角50%形成完美圆形
3. 中心指示器：
   - 圆形，直径16px
   - 填充白色，2px深灰（#374151）边框
   - 外阴影：0 2px 8px rgba(0,0,0,0.3)
   - 初始位置：色轮顶部（0度/红色区域）

### 5.4 交互要求

1. **拖拽逻辑**：
   - mousedown：检测鼠标是否在色轮半径内，是则开始拖拽
   - mousemove：计算鼠标相对于圆心的角度（Math.atan2），转换为色相值（0-360），更新指示器位置
   - mouseup：停止拖拽，触发颜色选择事件
2. **指示器限制**：始终保持在色轮半径内
3. **性能优化**：色轮仅在初始化时绘制一次，拖拽时仅更新指示器位置和颜色预览，不重绘整个色轮
4. **色轮只绘制一次**：在组件挂载时绘制，拖拽时仅重绘指示器，避免性能问题

### 5.5 附属元素

- **色轮下方**：当前选中颜色预览方块（40×40px，圆角8px，背景色=选中色） + HEX色值文字（14px，monospace字体，如#FF0000）
- **色轮右侧**（或下方）：色彩和谐规则下拉选择框
  - 宽度200px，高度36px
  - 选项：互补色、邻近色、三角色、矩形色、单色渐变
  - 默认选中"互补色"

### 5.6 事件通信

组件需通过自定义事件或回调函数向外传递：
- `color-selected`：携带`{ hex: "#FF0000", hue: 0, saturation: 100, lightness: 50 }`

## 6 色板展示组件

### 6.1 目标

展示当前激活的配色方案（5-6个色块），支持复制色值和锁定颜色。

### 6.2 布局要求

- 色块横向排列，flex布局，gap: 12px，justify-content: center
- 每个色块：宽度170px，高度120px，圆角12px

### 6.3 色块设计

每个色块的结构：
```
┌──────────────────┐
│  🔒         📋  │  ← 左上角锁定图标(仅随机模式显示)，右上角复制图标(hover显示)
│                  │
│                  │
│                  │
│                  │
│    #FF6B6B      │  ← 底部居中，白色文字，半透明黑底标签(圆角4px，padding 4px 8px)
└──────────────────┘
```

### 6.4 交互要求

1. **hover效果**：
   - 色块轻微放大：transform: scale(1.05)，过渡0.2s ease
   - 右上角复制图标显示（默认透明度0，hover时opacity: 1）
   - 色块阴影加深至`--shadow-hover`

2. **点击复制**：
   - 点击复制图标（或双击色块任意位置），调用`navigator.clipboard.writeText(hexValue)`
   - 复制成功后，色块中心短暂显示"✅ 已复制"提示：
     - 绝对定位居中，绿色背景（#10B981），白色文字，12px字号
     - padding: 6px 12px，圆角6px
     - 动画：0.3s淡入，保持0.5s，0.3s淡出消失

3. **锁定功能**（仅在随机生成模式下显示）：
   - 锁定图标默认灰色（未锁定），点击后变为金色（#FFD700）
   - 锁定状态：色块边框变为2px solid #FFD700
   - 锁定后该颜色不参与随机刷新

4. **选择微调**：
   - 点击色块（非复制图标区域）选中该颜色
   - 选中状态：色块边框变为2px solid #7C3AED，轻微发光效果
   - 选中后右侧微调面板绑定该颜色

### 6.5 状态

- 正常状态：5-6个色块显示当前配色
- 空状态：若配色方案为空，显示灰色虚线框+文字"暂无配色方案，请选择生成方式"
